C2D URL Options (NMS)

Connection
IP address or hostname
Username
Password
URLs
Following variables can be used inside URLs:
$numberNumber to dial.
$phoneAddressIP address or hostname
$phoneUsernameOptional: Username for authentication.
$phonePasswordOptional: Password for authentication.
Dial
Example: http://$phoneUsername:$phonePassword@$phoneAddress/command.htm?number=$number (SNOM 3x0)
Hangup
Example: http://$phoneUsername:$phonePassword@$phoneAddress/command.htm?RELEASE_ALL_CALLS (SNOM 3x0)
Phone Display
Example: http://$phoneUsername:$phonePassword@$phoneAddress/screen.bmp (SNOM 3x0)
